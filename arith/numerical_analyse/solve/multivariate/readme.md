### 超松弛迭代法
高斯-赛德尔迭代法的改进方法，引入了松弛因子w（记忆因子）

$$
x_i^{(k+1)}=(1-\omega)x_i^{(k)}+\frac{\omega}{a_{ii}}(b_i-\sum_{j=1}^{i-1}{a_{ij}x_j^{(k+1)}}-\sum_{j=i+1}^n{a_{ij}x_j^{(k)}})
$$


- w>1 超松弛迭代法 SOR 
- w=1 高斯-赛德尔迭代法
- w<1 低松弛迭代法
