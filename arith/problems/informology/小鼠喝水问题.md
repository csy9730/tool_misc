# 小鼠喝水
已知有1023瓶水，有一瓶有毒，有10只小鼠，如何测出有毒的那瓶？

设计观测矩阵
``` python

mat = []
N = 1024
M = 10
for i in range(1, N):
    s = bin(i)[2:]
    if len(s)<M:
        s= '0'*(M-len(s))+s
    s = [int(v) for v in s]
    mat.append(s)

# pick a number random
K = 555 

kk = [0 for i in range(1, N)]
kk[K-1]=1

import numpy as np
p = np.matrix(kk)*np.matrix(mat)
pp = p*np.matrix([[2**(M-i-1)] for i in range(M)])
k2 = int(pp)
print(K==k2)
```
