# 创维电视如何打开ADB


1. 首先进入“设置”-点击“本机信息”
2. 用遥控器方向键键入“上上下下左右左右”，注意：不是“上上下下左左右右”哦
3. 进入“工厂模式”的样子
4. 之后选择“高级设置”
5. 选择“ADB开关”，把它打开即可安装第三方引用


[https://www.znds.com/](https://www.znds.com/)


## 小米手机打开adb

- 连续点击Android版本/MIUI版本，激活开发者模式
- 连续点击内核版本，打开CIT测试页面
- 连续点击机身存储，可以打开 电话信息

激活开发者模式后，打开开发者模式菜单，可以打开USB调试模式。
