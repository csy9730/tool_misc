# 前言

要说玩游戏最舒服的时候，莫过于捧着手机躺在被窝里玩PC大作游戏，亦或是用5G网络在家外面玩电脑游戏，画质优秀，延迟丝毫不影响操作。目前各大厂商推出的云游戏服务固然不错，但昂贵的计时价格、参差不齐的云电脑配置、少得可怜的游戏数量，使得云游戏的最佳方案仍然是用自己的电脑当云游戏服务器，串流自己的电脑玩任何想玩的游戏。下面就介绍一下比较有名的几款串流软件，供大家参考。

# moonlight

目前最优秀的串流软件，串流效果好，功能丰富，支持虚拟手柄。无论是画质还是串流的稳定性，都秒杀其他软件，支持的分辨率也多，且客户端支持全平台，不管是安卓还是Windows还是mac还是linux甚至是破解后的Switch，都能使用这款软件来串流。要说唯一的缺点，可能就是只支持型号较新的N卡(例如gtx 960)，A卡用户还是接着往下看吧。另外就是跨局域网串流还需要再额外配置。

![串流软件汇总：用自己的电脑才是真正的云游戏 零成本](https://i0.hdslb.com/bfs/article/b4513d47c03b67cb7f307ca1aa637358147c45e5.png@942w_507h_progressive.webp)设置界面

官网：moonlight-stream.org

# parsec

最推荐小白使用的串流软件，配置方便，支持使用N卡、A卡以及英特尔核心显卡进行串流，也就是说对显卡没有要求！目前的串流质量已经相当接近parsec了，而且自带跨局域网串流功能，无需额外配置，还能邀请多个小伙伴一起来串流自己的电脑玩本地同屏多人游戏。最大的缺点是不支持虚拟按键，在手机上串流的话，需要自备实体手柄。

![串流软件汇总：用自己的电脑才是真正的云游戏 零成本](https://i0.hdslb.com/bfs/article/1f0a3af4564ec8bfc2e79ce5fc4493dea8d3bbaa.png@942w_594h_progressive.webp)电脑端



![串流软件汇总：用自己的电脑才是真正的云游戏 零成本](https://i0.hdslb.com/bfs/article/c075063424a33ca3eb1762fac15e3e39d636bc60.jpg@942w_425h_progressive.webp)手机端

官网：**parsec.app**

# fastlink

国产软件，也是完全免费，和parsec很相似，使用很方便。与parsec相比，缺点是画质略差、没有编解码选项，优点是支持虚拟按键，例如虚拟键盘按键或者虚拟手柄。

![串流软件汇总：用自己的电脑才是真正的云游戏 零成本](https://i0.hdslb.com/bfs/article/dbc9908826c4d15a66fe660d68c30952ead6dacd.jpg@942w_425h_progressive.webp)手机端

官网：**
**https://fastlink.icloud.cn/resource/

# steamlink

Steam出品，个人认为虽然使用也很方便，但是效果不如前面介绍的几款，个人有兴趣的话可以自行试用。没有pc版，因为PC版相当于集成在Steam中了。

# AMD Link

AMD官方出品，只支持A卡，局域网串流还行，跨局域网串流效果惨不忍睹，如果是广域网串流的话一般不推荐使用。

![串流软件汇总：用自己的电脑才是真正的云游戏 零成本](https://i0.hdslb.com/bfs/article/67c6dc6a8d3e38591e9afdbb55ad93da56bc6f72.png@942w_327h_progressive.webp)官网

官网：
https://www.amd.com/zh-hans/technologies/radeon-software-amdlink

# 总结

总的来说，如果你只需要局域网串流，那么建议N卡用户使用**moonlight**，A卡用户可以选其他的。如果需要用到跨局域网串流，又不想花功夫配置网络，那么建议选择**parsec**或者**fastlink**。
