# msys环境常用工具

linux下有很多常用工具，在msys下可用
* vim nano 文本编辑
* grep 字符串正则
* tree 文件目录显示
* zsh fish dash 终端工具
* tmux screen 终端窗口
* ldd,ntldd 动态依赖搜索
* curl,wget 下载器
* rsync 同步工具
* gnupg,ssh-keygen 密钥和加密
* sftp,ftp 文件传输
* git 版本管理
* openssh 加密终端
* dos2unix unix2dos 回车转换
* md5sum shasum uuidgen md5计算
* procps-ng
* inetutils
* openbsd-netcat
* mcrypt 
* pacman 包经理工具
* winpty 非常好用的终端转发工具
* gcc c语言编译工具
* cmake c项目组织工具
