### clover
Clover是一款将Windows资源管理器增加多标签功能的增强小软件。

它与Windows操作系统无缝集成，当打开“我的电脑”时，它能自动将窗口贴合到Clover的标签里，使用习惯与系统操作习惯一致，无需另外学习软件管理操作。
